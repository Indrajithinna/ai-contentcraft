# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-14

### Added
- Initial release of AI ContentCraft
- Multi-modal AI content generation platform
- Text generation using Google Gemini 2.5 Flash
- Image generation using Gemini 2.5 Flash Preview Image
- Audio generation using ElevenLabs API
- Multiple pre-built content templates
- Save and export functionality
- Comprehensive documentation

### Changed
- Migrated from Hugging Face to Gemini 2.5 Flash Preview Image
- Improved image quality and generation speed
- Enhanced error handling across all services

### Removed
- Hugging Face dependencies and services

## [0.0.0] - Initial Development

### Added
- Project scaffolding
- Basic React + TypeScript + Vite setup
- Initial component structure
